# AppleIntelWifiAdapter
苹果IO80211Controller调用

针对Apple未公开的AirPort程序进行的研究

目前基于10.15api

闲暇时间鼓捣。。。

## 使用方法，
xcode编译完成后
``` Bash
sudo chown -R root:wheel AppleIntelWifiAdapterV2.kext/
sudo kextload -v AppleIntelWifiAdapterV2.kext/
```
